1. Kapan kita harus menggunakan Array?
jika kita ingin menyimpan banyak data pada 1 variabel

2. Apa itu Property Array?
Property Array adalah fitur yang sudah disediakan oleh Javascript untuk memudahkan developer

3. Apa itu Method Array?
Method Array adalah sebuah fitur dari Javascript yang memudahkan kita dengan menyediakan function/method umum yang bisa kita gunakan.

4. Jelaskan apa perbedaan looping Array .forEach() dan .map()?
.forEach() adalah method untuk melakukan looping pada setiap elemen array sedangkan .map() melakukan perulangan/looping dengan membuat array baru.
